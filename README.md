# UPI Fraud Detection System

This is a Machine Learning based web application that detects fraudulent UPI transactions.

## Features
- Fraud detection using ML model
- Web interface for prediction
- Fast and accurate prediction
- Built using Python, ML, and Web technologies

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit / FastAPI
- HTML, CSS

## How to Run

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run app.py

or

python app.py

## Project Structure

Main_Codes → contains main application code  
Documents → contains report and presentation
